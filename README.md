# MargenVivo — control de ventas y margen para PYMEs

Prototipo funcional. Dos usuarios:
- **Vendedora:** registra ventas eligiendo el SKU de una lista y ve el **margen en vivo** + alerta si baja del piso.
- **Administrador:** login, dashboard semanal (KPIs, ranking de vendedoras, margen por categoría, alertas).

**Demo en vivo:** https://dreeromer.github.io/margenvivo-demo/
